# 『マヨイビト』DCG v0.7.0

v0.7 で、スマートフォンのゲームらしい**メインハブ画面**を新設しました。
ホーム／カード／対戦／ショップ／その他の5つのタブから各機能へ進めます。

## v0.7 で新しくなったところ

- 5つのタブを持つメインハブ画面（共通の一枚絵・プレイヤー情報・ミッション枠）
- プロフィール／お知らせ／設定／ヘルプ／用語集／よくある質問／クレジットの各画面
- データの書き出し・読み込み・初期化（データ管理）
- 対戦・カード・デッキなど既存の機能へ、新しい導線からつながるように
- 文字サイズとタップ領域を見直し、小さな画面でも読みやすく押しやすく

準備中の機能（ショップ・オンライン・ストーリーなど）は、
画面や入口だけ用意して「準備中」と案内しています。

---

自作トレーディングカードゲーム『マヨイビト』を、ブラウザで遊べるようにしたものです。

**スマートフォンの縦画面**を基準に作っています。横向きにすると注意が出ます。
HTML・CSS・JavaScript だけで動きます。外部の枠組みは使っていません。

いまは**開発とプレイテストのための最小実装版**です。

---

## 遊べるもの

| | 内容 |
|---|---|
| **チュートリアル** | 基本編と実践編。操作しながらルールを覚えます |
| **CPU対戦** | CPUを相手にひとりで遊びます。難易度は5段階 |
| **ひとり回し** | 1台の端末で、両方のデッキを交互に操作します |
| **CPU観戦** | CPU同士の対戦を自動で流します（開発者用モードの中） |

対戦では**シードを固定**でき、同じ展開をもう一度再現できます。

### CPUの難易度

| 難易度 | 目安 |
|---|---|
| 弱 | カードゲームに不慣れな人向け |
| 中 | 基本的な行動を行う標準的なCPU |
| 強 | 盤面や手札を考えて効率的に動く |
| エキスパート | 高度な判断を行う、本気の対戦用 |
| 理不尽 | CPUがこっそりイカサマします。公平な勝負ではないおまけ |

はじめてなら**弱**から、ルールを覚えたら**強**を試してください。

### そのほか

- **カード一覧**（所持枚数つき／検索・絞り込み・並べ替え）
- **デッキ一覧**と**デッキ編成**（自作デッキの保存、公式デッキのコピー）
- **ローカルセーブ**と、その**書き出し・読み込み**
- 設定（CPUの速さ、演出の速さ、効果音、レーンの左右反転など）
- 対戦結果とエラー情報のコピー

## チュートリアル

操作しながらルールを覚える、正式なチュートリアルです。
**実践編は、基本編をクリアすると解放**されます。

### 基本編

はじめから対戦を始めて、ひととおりの流れを体験します。

- マリガン
- 勝利条件
- カードの登場
- 追跡
- 襲撃
- フィールド効果
- ロストとトラッシュ
- 人間を補充して敗北を避ける

### 実践編

試合の途中の盤面から始めて、勝ち切るまでを体験します。

- グッズ
- イベント
- 気力の使い方と、次のターンへの持ち越し
- カードを使う順番
- 高コストのカード
- 蓄積ダメージ
- フィールド効果
- ロスト上限を意識した勝ち方

終わると、弱いCPUとの対戦をすすめる画面が出ます。

## 操作方法

| したいこと | 操作 |
|---|---|
| カードを登場・使用・装備する | 手札から**ドラッグ** |
| 追跡する | 自分の怪異から相手の人間へ**ドラッグ** |
| カードの詳細を見る | **タップ**で簡易表示、**長押し**で拡大表示 |
| メニューやボタン | **押す** |

手札は、**上へなぞると大きく広がります**。ドラッグでカードを出せるのは、広げているときだけです。

演出の最中や相手のターン中でも、カードの詳細は見られます。

## デッキ構築

モード選択の「カード」→「デッキ一覧」から作れます。

| | |
|---|---|
| メインデッキ | **40枚** |
| フィールド | 別枠（メインの40枚には数えません） |
| 同名カード | **4枚**まで |
| 自作デッキ | **10個**まで |

- フィールドを選ぶと、**対応する0コストの主人公が自動で入ります**（削除できません）
- 陣営を**混ぜられます**。ヨマモリ村と黒薔薇の館のカードを1つのデッキに入れられます
- **未完成のままでも保存できます**。使えないデッキは一覧で暗く表示され、理由を確認できます
- 公式デッキは編集・削除できません。**コピーしてから**手を入れてください
- エースカードは長押しで設定します。設定しなくても対戦できます

CPU対戦とひとり回しで自作デッキを使えます。CPU側と観戦は公式デッキのみです。

## 起動方法

インストールは要りません。**`index.html` をブラウザで開くだけ**です。

外部からファイルを取ってくる作りではないので、ローカルサーバーは不要です。

---

## テスト方法

Node.js が必要です。プロジェクトの直下で実行してください。

```bash
# 全部まとめて
for f in tests/*-tests.js; do node "$f" || echo "失敗: $f"; done

# よく使う個別のもの
node tests/rules-fingerprint-tests.js    # ルール処理が変わっていないか
node tests/battle-deck-tests.js          # AIの指し手が変わっていないか
node tests/release-tests.js              # 配布前の点検
node tests/release-metadata-tests.js     # 版の表記がそろっているか
node tests/tutorial-advanced-script-tests.js   # 実践編を最初から最後まで通す
```

構文の検査だけなら、こうです。

```bash
for f in js/*.js; do node --check "$f" || echo "構文エラー: $f"; done
```

**実機での確認項目は `TESTING.md`** にまとめてあります。
見た目や操作感は自動テストでは判断できないので、そちらを使ってください。

## 保存データ

ブラウザの `localStorage` に保存しています。

保存されるもの：

- 設定
- 所持カード
- 自作デッキ
- チュートリアルのクリア記録
- 前回選んだデッキや難易度

**ブラウザのデータを消すと、一緒に消えます。**
設定画面から**書き出し**ておくと、ファイルとして残せます。
大事なデッキができたら、書き出しておくことをおすすめします。

## 開発者向け

- `TESTING.md` … 実機で確かめる項目の一覧
- `tests/` … 自動テスト
- `tools/clear-card-corners.py` … カード画像の四隅の白を透明にする道具

**カード画像を差し替えたら、`tools/clear-card-corners.py` を実行してください。**
実行しないと、カードの四隅に白が出ます。
