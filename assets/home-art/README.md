# 一枚絵（ホームの共通アート）の作り方

メインハブの背景いっぱいに出る絵です。起動のたびランダムで選ばれ、
一定時間ごとに次の絵へ切り替わります（0.5秒のクロスフェード）。

## 仕様

| 項目 | 内容 |
|---|---|
| サイズ | **1080 × 1920 px**（9:16） |
| 形式 | **WebP** |
| 置き場所 | `assets/home-art/` |
| 表示 | 横幅いっぱい・**上端そろえ**（`background-size: cover` / `background-position: center top`） |

## テンプレート

| ファイル | 用途 |
|---|---|
| `_template-guide.png` | **下敷き用**。いちばん下のレイヤーに置いて、その上に描く |
| `_template-preview.png` | **確認用**。描いた絵の上に重ねて、隠れ具合を見る |

どちらも `python3 tools/make-art-template.py` で作り直せます。
UI の位置を変えたときは、作り直せば数値が自動で追従します。

## 構図のきまりごと

```
   0px ┌─────────────────────┐
       │ プレイヤー情報      通貨 │  UIが重なる
 260px ├─────────────────────┤
       │                         │
       │    ここに顔・見せ場     │  ← 安全範囲（高さ972px）
       │                         │
1232px ├─────────────────────┤
       │  バナー・ミッション枠   │  UIが重なる
       │  タブ固有パネル         │
1800px ├─────────────────────┤
       │      メインナビ         │
1920px └─────────────────────┘
```

- **顔と見せ場は 260〜1232px の帯に収める**。中央よりやや上が収まりよし
- 上下はUIが重なるので、細かい描き込みをしても見えない
- 下半分は暗めのグラデーションが掛かる（文字を読みやすくするため）ので、
  明るい要素を置いても沈む

## 追加のしかた

1. 1080×1920 で描く
2. WebP で書き出して `assets/home-art/` に置く
3. `js/v07-hub-data.js` の `V7_HOME_ART` に1件足す

```javascript
{
  id: 'elise_02',                                  // 重複しない名前
  characterName: 'エリーゼ',                        // 通知に出る呼び名
  displayName: '屋敷の令嬢 エリーゼ',                // 起動時の1回だけ上部に出る表記
  imagePath: 'assets/home-art/elise_02.webp',
  enabled: true,                                   // false にすると出番なし
},
```

`enabled: false` にすれば、消さずに一時的に外せます。
